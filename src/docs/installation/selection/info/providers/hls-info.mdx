Embed video content into documents via the native video element. This provider enables
streaming video using the HTTP Live Streaming (HLS) protocol.

[HLS isn't widely supported yet][can-i-hls], but we use the popular [hls.js][hlsjs] library to
ensure it works anywhere [Media Source Extensions][can-i-mse] (MSE) are supported, which
accounts for ~96.42% of users tracked on caniuse.

[can-i-hls]: https://caniuse.com/?search=hls
[hlsjs]: https://github.com/video-dev/hls.js
[can-i-mse]: https://caniuse.com/mediasource
