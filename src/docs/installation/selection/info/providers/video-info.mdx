You can use this provider to build a video player. This provider uses the native
`<video>` element to support [video codecs][codecs] such as AV1, AVC (H.264), VP9, etc.

Do note, the type of video codec supported is completely dependent on the browser so use the
prior link as reference to see what's supported.

[codecs]: https://developer.mozilla.org/en-US/docs/Web/Media/Formats/Video_codecs
